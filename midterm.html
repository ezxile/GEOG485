Question 2

What combination of OGC Service and Request would you use to perform the following?

    1. Determine the spatial extent of an available layer from a Web Map Service:
    SERVICE=WMS&REQUEST=GetCapabilities

    2. Obtain a list of coverages from a Web Coverage Service:
    SERVICE=WCS&REQUEST=GetCapabilities

    3. Determine what file formats are supported by a Web Feature Service for the delivery of available data types:
    SERVICE=WFS&REQUEST=GetCapabilities

    4. Retrieve a map image from a Web Map Service:
    SERVICE=WMS&REQUEST=GetMap

    5. Retrieve data from an available coverage from a Web Coverage Service:
    SERVICE=WCS&REQUEST=GetCoverage
