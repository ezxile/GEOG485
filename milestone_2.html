Milestone 2

Question 1

From examining the display in your web browser and the structure of the source code in the page, what effect (if any) does the white space (i.e. tabs, blank lines, multiple spaces) have on what is displayed in the browser?

White space does not make a difference in code unless body texts are defined, then white space is shown.

Question 2

How are the (<h1>) and (<h2>) elements from the source code displayed differently in the browser?

Unless style for these elements are defined, the only difference between headings is font size. (<h1>) headings are larger because they are more significant than (<h2>) headings.

Question 3

What type of element would you use to create additional list elements in either the “topic” or “data type” lists on the page.

You can use the (<li>) tag to define list terms. Ordered lists (<ol>) and unordered lists (<ul>) define the order that these line items appear in.
